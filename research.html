Topological VQE
GitHub 2023 – Present

Topological Materials via VQE

VQE framework for identifying topological superconductor candidates via the Kitaev chain. Includes real-space BdG Hamiltonian construction, Jordan-Wigner qubit mapping, and topological invariant computation (winding numbers, entanglement entropy). Probing Majorana zero modes and topological phase transitions including effects of disorder.

More details

Built on Qiskit 2.x with SparsePauliOp and BackendEstimatorV2. The pipeline constructs the real-space BdG Hamiltonian with optional on-site disorder, maps it to qubit operators via Jordan-Wigner transformation, and runs VQE with hardware-efficient ansätze. Topological characterization includes winding number computation, entanglement entropy across bipartitions, and finite-size scaling analysis near the phase boundary at |μ| = 2t.

Key tools: Qiskit 2.x, NumPy, SciPy, Matplotlib, SLURM/HPC

GitHub 2024 – Present

MZM Visibility

CNN pipeline predicting Majorana zero mode visibility from nanowire conductance data. Uses ResNet-18 architecture with PyTorch for training and inference on simulated datasets. Includes data generation, preprocessing, and evaluation with quality gate verification.

More details

Trains a modified ResNet-18 on simulated nanowire conductance maps (G(V_bias, V_gate)) to predict a continuous visibility score for Majorana zero modes. The pipeline includes HDF5 data loading, train/val/test splitting with leakage prevention, mixed-precision training on A100 GPUs via SLURM, and evaluation with reconstruction plots comparing predicted vs. ground truth visibility.

Key tools: PyTorch, NumPy, h5py, Matplotlib, SLURM/HPC

SC Quantathon v2 (2025)

QSVM, QNN, and Cirq-based quantum kernels for tornado prediction, developed for the SRNL Quantum Computing Challenge. Combines classical ML with quantum feature maps to explore quantum advantage in weather pattern classification.

More details

Benchmarked quantum kernel methods against classical baselines (XGBoost, SVC, logistic regression) on NOAA storm report data. Built Cirq-based quantum feature maps with ZZFeatureMap and custom entangling layers, compared QSVM and QNN classification accuracy, and analyzed quantum kernel alignment to assess quantum advantage potential.

Key tools: Cirq, scikit-learn, XGBoost, Pandas, Matplotlib

Circuit Synthesis
GitHub 2026

iQuHACK 2026

Reed-Muller decoding-based Clifford+T circuit optimizer (rmsynth). Developed at MIT’s annual quantum hackathon for efficient quantum circuit synthesis, targeting reduced T-gate counts and circuit depth.

More details

Implements the rmsynth algorithm for decomposing arbitrary single-qubit unitaries into Clifford+T gate sequences using Reed-Muller code structure. Leverages pygridsynth for grid operator search and applies RM decoding to minimize T-gate count, improving circuit depth for fault-tolerant quantum compilation.

Key tools: Python, NumPy, SciPy, pygridsynth

iQuHACK 2025

Cat qubit simulation for the Alice & Bob open quantum systems challenge. Explored bosonic error correction codes and noise-biased qubit architectures at MIT’s annual quantum hackathon.

More details

Simulated cat qubit dynamics using Dynamiqs for the Alice & Bob open quantum systems challenge. Modeled two-photon driven dissipative stabilization of cat states, analyzed bit-flip vs. phase-flip error asymmetry, and benchmarked decoherence rates against standard transmon qubits.

Key tools: Dynamiqs, Python, NumPy, Matplotlib

GitHub 2025

NeurotiQ (NYUAD 2025)

Hybrid QNN/QSVM system for EEG analysis built with PennyLane, featuring a Streamlit web interface. Developed at the NYU Abu Dhabi Quantum Hackathon for neurological disorder detection using quantum-classical hybrid models.

More details

Built a full-stack quantum ML application for EEG-based neurological disorder detection. The quantum backend uses PennyLane with StronglyEntanglingLayers and AmplitudeEmbedding for both QNN and QSVM approaches. Classical preprocessing includes PCA, StandardScaler, and SMOTE for class imbalance. The Streamlit frontend provides real-time inference and visualization of classification results.

Key tools: PennyLane, TensorFlow, scikit-learn, Streamlit
